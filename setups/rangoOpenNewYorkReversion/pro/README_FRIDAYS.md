# Rango Open New York Reversión (FRIDAYS ONLY)

Este robot es una versión especializada del setup de reversión de Nueva York, configurado para operar **exclusivamente los viernes**. 

### ¿Por qué operar solo los viernes?
Tras el análisis estadístico del periodo 2024-2025, se determinó que el viernes es el día con mayor ventaja probabilística para este setup específico:
- **Winrate:** 42.31% (vs 32% promedio semanal).
- **Profit Factor:** 2.04 (vs 1.35 promedio semanal).
- **Esperanza Matemática:** 0.69 R por trade (vs 0.25 R promedio semanal).

### Configuración Técnica
- **Días permitidos:** Viernes.
- **Magic Number:** 12348.
- **Parámetros:** Idénticos a la versión v3.0 PRO (SL 6000, TP 18000, Exclusión de Rango activa).

### Uso
Este experto está diseñado para cuentas que buscan maximizar el Profit Factor reduciendo el "drawdown" de los días menos rentables de la semana (como los jueves).
